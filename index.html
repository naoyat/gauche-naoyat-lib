Function: assq-value key assoc-list
Function: assv-value key assoc-list
Function: assoc-value key assoc-list

fillがリストの場合、fillのパターンを繰り返して埋めた長さlenのリストを生成します。

2.3 naoyat.printf

C言語風なprintf/sprintfを提供します。 デバッグに使うのがメインかと思いますが、formatが嫌いな人は日常的な用途にも使うかもしれません。

独自仕様として、%bで２進表記が得られるようになっています。（嫌なら該当部分をコメントアウトしてください）

※printf/sprintf はslibにもあります。c-wrapperでも行けると思います。

Function: printf fmt args ...
Function: sprintf fmt args ...
 
(printf "%03d" 7) ⇒ "007"
(printf "%x" 15) ⇒ "f"
(printf "|%5s,%-5s|" "abc" "def") ⇒ "|  abc,def  |"
(printf "%g" 1/2) ⇒ "0.5"

2.4 naoyat.math

(naoya_tが)よく使う数学的関数をまとめています。 libm の関数をc-wrapper経由で直接呼び出すものは naoyat.math.h をご参照ください。

Function: != x y

(x != y)を真偽値で返します。

Function: square x

xの２乗を返します。

Function: factorial x

xの階乗 (x!) を返します。

Function: C n m

nCm を返します。

Function: Σ list

list の要素の和を返します。

Function: Π list

list の要素の積を返します。

Function: log2 x

底を2とする対数を返します。

Function: log10 x

底を10とする対数を返します。

Function: log16 x

底を16とする対数を返します。

Function: interpolate x0 x1 t

実数値x0,x1,tに対し、t=0の時の値をx0、t=1の時の値をx1とした線形補間(内挿/外挿)を行います。 tは[0,1]に含まれなくても構いません。

Function: inf-filter x [limit]

xがNaNの場合に#fを返す以外は (clamp x -limit +limit) と同義です。グラフを描くときに使っています。

Function: expt* x y

expt同様 x^y を返しますが、x=0の時に0を返します。

Function: logistic-sigmoid a
Function: σ

ロジスティック・シグモイド関数 1/(1 + exp(-a))

Function: logit

ロジット関数 log( p/(1-p) )

2.4.1 naoyat.math.random

乱数

Function: random-integer min max

Mersenne-Twisterを用い、[min,max]の範囲の整数をランダムに生成します。

Function: make-uniform-rand min max

Mersenne-Twisterを用い、[min,max]の範囲の実数をランダムに生成する関数を返します。

Function: box-muller-rand

Box-Muller法を用いガウス分布（正規分布）乱数を生成します。

Function: make-nd-rand μ σ

平均μ,分散σをもつガウス分布（正規分布）乱数を生成する関数を返します。

内部でBox-Muller法を用いています。

Function: make-nd2-rand μ Σ

平均μ,共分散行列Σをもつ二次元ガウス分布乱数を生成する関数を返します。

内部でBox-Muller法を用いています。

2.4.5 naoyat.math.prime

素数

Function: primes-until n

n以下の素数をリストで返します。キャッシュがあればそれを利用します。

Function: prepare-primes-until

n以下の素数を求め、キャッシュしておきます。

Function: prime? n

nが素数かどうかを判定し、真偽値を返します。

Function: prime-factors
Function: prime-factors*
Function: prime-factor-table
Function: largest-prime-factor n

nの素因数のうち最も大きいものを返します。

Function: nth-prime n

n番目の素数を返します。

Function: the-sum-of-the-primes-below-n

nまでの素数の和を返します。

Function: longest-sum-of-consecutive-primes n

nまでの数で、連続する素数の和である数のうち、最も多くの素数から成るものを返します。

Function: n-digit-primes n

n桁の素数をリストで返します。

Function: fast-prime? n [times]

Miller-Rabin法を用いた素数判定を行い、真偽値を返します。 但しnが2の場合は#tを、2以外の偶数の場合は#fを無条件に返します。 Miller-Rabin法の繰返し回数はtimesで指定します。（省略した場合は3）
